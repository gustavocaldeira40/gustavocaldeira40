## About

I build end-to-end digital products, from UI architecture and Figma handoff to APIs, authentication, databases, integrations, deployment, and production maintenance.

My strongest zone is modern front-end engineering with full-stack ownership: React, Next.js, TypeScript, Node.js, NestJS, clean UI systems, reliable API contracts, motion-rich interfaces, and pragmatic delivery for real business workflows.

- 5+ years of experience across front-end and full-stack roles, including JSMX, Orbits Technology, Startec, Vsi, and Eduzz
- 15+ documented projects across web apps, APIs, dashboards, automation, integrations, games, academic systems, and freelance work
- 6+ deployed projects available to test, including real client websites, production interfaces, and technical challenges
- 4+ real clients supported through software delivery, websites, integrations, automation, infrastructure, and IT solutions
- Currently building **Alpharm KIDS**, a gamified literacy platform with Next.js and NestJS
- Preparing for an English immersion program in Canada and open to international opportunities
- Focused on polished interfaces, scalable architecture, integrations, animation, performance, and product thinking
